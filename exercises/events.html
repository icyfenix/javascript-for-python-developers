Book List Part 3

The last phase of the book list! From the previous exercise, we're going to add buttons and inputs that allow you to have a fully-featured application in JS!
  • Start by adding 3 <input> tags, one for each text-based property of a book. Also add 2 inputs for a radio button group, to indicate whether or not it's been read. (scroll down to where it says radio button)
  • Add a button to the page, and then with javascript, use the .onclick property of the button ( element.onclick = function(){ //things } ). The function should do these things:
    • From inside the function, find the <input> tags
    • Check their values, assign them to variables
    • Create a Book() object from the values
    • .add() the book to the BookList
  • Make sure there is code in the .add() function that updates the <ul> tag, appending a new <li> for the new book.
  • Bonus: Add another button that allows you to click the cover image of a book and toggle it's .read value.
